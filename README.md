# 🌐 Internet Speed Test (Python GUI)

A simple **Internet Speed Test application** built using **Python, Tkinter, and Speedtest library**.
This app allows users to check their **download** and **upload speed** with a single click in a graphical interface.

---

## 🚀 Features

* 📊 Check **Download Speed**
* 📤 Check **Upload Speed**
* 🖥️ Simple and user-friendly GUI (Tkinter)
* ⚡ One-click speed test
* 🎯 Real-time result display

---

## 🛠️ Technologies Used

* **Python**
* **Tkinter (GUI)**
* **Speedtest-cli**

---

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/internet-speed-test.git
cd internet-speed-test
```

### 2. Install required library

```bash
pip install speedtest-cli
```

---

## ▶️ How to Run

```bash
python your_file_name.py
```

---

## 🧠 How It Works

* The app uses the **Speedtest API** to measure internet speed.
* When you click the **CHECK Speed** button:

  * It connects to the nearest server
  * Calculates:

    * Download speed
    * Upload speed
* Results are displayed in the GUI.
